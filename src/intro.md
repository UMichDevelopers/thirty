# Introduction

This guide aims to provide general information for incoming students of the
Class of 2030 of the University of Michigan.

## Community

Please join us through either (or both) of these methods!

- [Discord Server](https://discord.gg/bUYQyPjAVC)
- [`##umich-30-general`](https://web.libera.chat/##umich-30-general)
  on [Libera.Chat](https://libera.chat)

## Using this guide

It's basically a book. On the top left you should see a hamburger menu;
opening it presents you with the table of contents.

If you wish to edit anything, choose one of these options:
- Click the pencil on the top right; follow GitHub's instructions to edit the
  page and create a pull request.
- [Send patches](https://git-send-email.io/) to `runxiyu@umich.edu`.
- Tell us in any of the community spaces.
